KL Divergence

Detect distribution shift between current and baseline model outputs

WHAT IS KL DIVERGENCE?

Distribution Shift Detection

KL(P‖Q) measures how much the current distribution P has drifted from the baseline Q. High KL = model behavior has changed significantly. We also compute Jensen-Shannon divergence (symmetric, bounded 0–1).

KL(P‖Q) = Σ P(x) · log₂(P(x) / Q(x))

HOW FORECASTING WORKS

Holt's Double Exponential Smoothing

We apply trend-aware exponential smoothing to your entropy history, then extrapolate forward N steps. If the forecast crosses the critical threshold, you get an early warning with the exact step count.

Level(t) = α·H(t) + (1-α)·(Level(t-1) + Trend(t-1))

Calibration Scorer

Measure Expected Calibration Error (ECE) — does model confidence match real accuracy?

WHAT IS CALIBRATION?

Expected Calibration Error (ECE)

A perfectly calibrated model that says "I'm 80% confident" should be correct 80% of the time. ECE measures the gap between stated confidence and actual accuracy. ECE = 0 is perfect; ECE > 0.2 is severely miscalibrated.

ECE = Σ (|Bₘ|/n) · |acc(Bₘ) − conf(Bₘ)|

How It Works

Step-by-step methodology walkthrough

01
INPUT

Confidence Scores

A model's confidence score (0–1) represents certainty about its output. Values near 0 or 1 = high certainty. Near 0.5 = maximum uncertainty.

P(output) ∈ [0.0, 1.0]
▼
02
COMPUTATION

Shannon Entropy

Binary Shannon entropy converts raw probability to uncertainty measure. Entropy 0 = fully deterministic. Entropy 1 = completely random.

H(p) = −(p·log₂p + (1−p)·log₂(1−p))
▼
03
DRIFT DETECTION

Rolling Window

Entropy readings stored in rolling window. Monotonic increase and drift velocity identify genuine model degradation vs noise.

drift = H(t) − H(t−2)
▼
04
EARLY WARNING

Tipping Classification

Three alert states give engineers time to retrain before production failure.

✅ SAFE⚠️ WARNING🔴 CRITICAL
▼
05
OUTPUT

Stability Score & Insight

Final Stability Score (0–100) from average entropy. Insight panel gives human-readable recommendation with action items.

Stability = (1 − avg_entropy) × 100
